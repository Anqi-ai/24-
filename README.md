# 小專題題目:算24點
## 組別號碼:17
## 系級班 級:資工1B
## 成員:李安淇、黃 榆棻、林倢妤
## 遊戲規則:
- 系統隨機產生4個數字(模擬撲克牌)
- 數值規則:
 - A = 1
 - 2 ~ 10 =原數字
 - J = 11 ， Q = 12 ， K = 13
- 玩家需使用:
 - 加法(+)
 - 減法(-)
 - 乘法(*)
 - 除法(/)
 - 括弧(( ))
 ### 限制條件
 - 每個數字只能使用一次
 - 不可使用未提供的數字
 - 不可除以0
 - 必須剛好算出24
 ---
 ## 遊戲流程:
 1. 隨機產生4個數字
 2. 玩家輸入運 算式
 3. 系統判斷:
    - 是否使用正確數字
    - 是否符合運 算規則
    - 結果是否為24
 4. 顯示結果(成功 / 失敗)
